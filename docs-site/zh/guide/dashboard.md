# 控制台

FerrumKV 内置了 Web 控制台 —— 无额外依赖，也无需单独部署服务。默认监听 `:6381`
（可通过 `--dashboard-addr` 修改，或设为 `off` 关闭）。

## 功能

| 功能 | 说明 |
|---------|-------------|
| Key 浏览器 | 通配符搜索（`user:*`、`session?`）、分页列表、TTL 徽章 |
| 就地编辑 | 在浏览器中查看、编辑、设置 TTL 并删除 key |
| 实时指标 | key 数量、内存、命中率、淘汰策略 —— 每 2 秒自动刷新 |
| 命令控制台 | 执行任意 RESP 命令（`GET`、`SET`、`INFO` …），带语法高亮与自动补全 |
| 命名空间概览 | 未选中 key 时展示可点击的命名空间分布网格 |
| 主题切换 | 在暗色与 macOS 原生亮色主题间切换（按浏览器记忆） |

## 打开方式

```bash
# 默认地址为 127.0.0.1:6381
open http://127.0.0.1:6381
```

控制台与 RESP 服务共享同一个 `KvEngine` 实例，因此在界面中的修改会立即对连接的客户端可见。
